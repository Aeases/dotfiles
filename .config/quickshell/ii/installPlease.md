# Required Packages for Quickshell (ii)

To fix the missing icons and restore full functionality, please install the following packages using `yay`:

## Essential for Icons and UI
- `ttf-material-symbols-variable-git` (or `ttf-material-symbols-variable`)
- `ttf-readex-pro`
- `ttf-rubik-vf`
- `ttf-twemoji`
- `ttf-jetbrains-mono-nerd`
- `otf-space-grotesk`

## Essential for Features
- `syntax-highlighting` (for AI chat code blocks)
- `matugen` (for material color generation)
- `cliphist` (for clipboard manager)
- `libqalculate` (for math search/calculator)
- `imagemagick` (for thumbnail generation)
- `fuzzel` (for app launcher fallbacks)
- `playerctl` (for media controls)

## Command
```bash
yay -S ttf-material-symbols-variable-git ttf-readex-pro ttf-rubik-vf ttf-twemoji ttf-jetbrains-mono-nerd otf-space-grotesk syntax-highlighting matugen cliphist libqalculate imagemagick fuzzel playerctl
```

Note: If `ttf-material-symbols-variable-git` is unavailable, you can use the stable version `ttf-material-symbols-variable` from the extra repository.
